# Tistory Skin Starter (Shadcn 느낌)

이 폴더는 티스토리 스킨에 바로 붙여넣을 수 있는 기본 패키지입니다.

## 파일 구성

- `skin.html`: 티스토리 HTML 영역에 붙여넣을 템플릿
- `style.css`: 티스토리 CSS 영역에 붙여넣을 스타일

## 적용 순서

1. 티스토리 관리자 > 꾸미기 > 스킨 편집 > HTML 편집으로 이동
2. HTML 탭 전체를 `skin.html` 내용으로 교체
3. CSS 탭 전체를 `style.css` 내용으로 교체
4. 저장 후 블로그 새로고침

## 참고

- 주신 기존 스킨처럼 파일 업로드 기반(`images/...`) 스크립트를 필수로 쓰지 않는 구조라서, 우선 복붙만으로 동작하도록 구성했습니다.
- 기존 HTML/CSS의 모든 고급 기능(모바일 확장 메뉴, iScroll, legacy dropdown JS)은 포함하지 않았습니다.
- 필요하면 다음 단계에서 기존 기능을 하나씩 이 스타일로 이식해드릴 수 있습니다.
