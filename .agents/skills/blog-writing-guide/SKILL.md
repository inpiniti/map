# 블로그 글작성 실전 스킬 (트렌드+SEO 기반)

이 스킬은 최신 트렌드(trend-2026/SKILL.md)와 SEO 실전 가이드(seo-guide/SKILL.md)를 기반으로 블로그 글 주제를 선정하고, 구글/유튜브 검색 결과를 분석·요약해 고품질 블로그 글을 작성하는 방법을 안내합니다.

## 0. 블로그 글작성 원칙

- seo-guide/SKILL.md의 블로그 글작성 원칙을 참고합니다.
- 원칙이라기보다 seo 에 검색이 잘될수 있도록 하는 가이드라인이라고 생각하면 됩니다.
- 검색이 안되는 글은 아무리 잘써도 소용이 없으니까요.

## 1. 주제 선정

- trend-2026/SKILL.md의 TOP 트렌드 참고해 주제를 선정합니다.
- 검색 의도, E-E-A-T, 최신성, 실질적 가치, 실전 수요를 고려합니다.

## 2. 구글링(웹 검색) 리서치

- 선정한 주제로 구글에서 검색해 상위 10개 이상의 글을 확인합니다.
- 각 글의 핵심 주장, 실전 팁, 공통점/차이점, 최신 정보, 실제 사례 등을 요약합니다.
- 신뢰도(E-E-A-T), 출처, 최신성, 실질적 가치가 높은 정보를 우선 반영합니다.

## 3. 유튜브 검색 리서치

- 동일 주제로 유튜브에서 10개 이상의 영상을 검색합니다.
- 각 영상의 주요 내용, 실전 노하우, 사례, 트렌드, 시청자 반응 등을 요약합니다.
- 영상에서만 얻을 수 있는 실제 경험, 최신 트렌드, 실전 팁을 적극 반영합니다.

## 4. 블로그 글 작성

- 구글/유튜브 요약 내용을 바탕으로, blog-writing-guide/SKILL.md의 글쓰기 원칙(검색 의도, 구조, 키워드, E-E-A-T, 독창성, 실질적 가치 등)을 적용해 글을 작성합니다.
- 도입부에 핵심 요약, 본문에 실전 정보/사례/비교/팁, 결론에 요약 및 실천 가이드 제시
- 표, 리스트, 인용, 실제 경험 등 다양한 형식 활용
- AI/자동화 활용 시 투명하게 공개, 독창적 분석/경험 추가

## 5. 최종 점검

- SEO 체크리스트(제목, 메타, 구조, 키워드, 내부/외부 링크, 이미지 alt 등) 점검
- 최신성, 신뢰성, 독창성, 실질적 가치, 사용자 만족도 재확인
